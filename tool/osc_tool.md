# OSC Tool

효율적인 Open Source Compliance 활동을 위하여 소스 코드 상 Open Source License를 자동으로 분석해주는 툴뿐만 아니라 프로젝트별 Open Source를 관리할 수 있도록 도와주는 툴 등 많은 유용한 툴들이 존재합니다. 

이 중 Open Source로 공개되어 있어 누구나 자유롭게 사용할 수 있는 툴들을 소개합니다.
